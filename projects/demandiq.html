DemandIQ

Retail Demand Forecasting & Inventory Analysis

A production-style demand forecasting platform processing over 1M+ retail records, achieving 94.9% R2 using Random Forest. Engineered an end-to-end ML pipeline integrating automated preprocessing, feature engineering, model selection, forecasting, and interactive KPI dashboards.

Timeline
2 Months (2025)
Role
Data Science Intern
Languages
Python
Target Audience
Store Managers
GitHub Repository

The Problem

Retail stores suffer huge financial leakages due to inefficient stock allocations. Over-stocking ties up valuable business capital in slow-moving items, while under-stocking popular products leads directly to missed revenue and negative customer loyalty reviews.

Store managers often rely on basic historical averages to guide restocking. However, they fail to take into account underlying features like seasonal sales variations, promotion spikes, and changes in buyer clustering distributions.

The Solution

DemandIQ handles these bottlenecks through a modular retail modeling pipeline. Utilizing Scikit-learn, the application parses past transaction histories and fits seasonal regression algorithms.

The platform exposes a clean dashboard visualizing low-stock critical events, forecasting daily sales demand with confidence boundaries, and generating automated, natural recommendations detailing which products need immediate restocking.

Process Architecture

Key Features

  • Demand Forecasting: Predicts daily sales trends 14 days in advance using Random Forest.
  • Inventory Analysis: Monitors active stock lines and flags items approaching critical out-of-stock metrics.
  • Customer Segmentation: Separates buyers into value categories to target promotional discounts.
  • AI Recommendations: Automatically alerts managers when stock levels drop below optimal thresholds.

Tech Stack

Python Streamlit Pandas Plotly Scikit-learn XGBoost Random Forest

Key Challenge

Managing noisy transactional dataset extracts. Raw logs contained significant anomalies (e.g. bulk sales returns, system failures representing missing data). Resolved this by engineering a custom preprocessing outlier filter using local outlier factor algorithms.

Outcome & Impact

Developed as a production-style demand forecasting platform, the system successfully processed over 1M+ retail records, achieving a 94.9% R2 accuracy score using Random Forest. The end-to-end ML pipeline ensures efficient preprocessing, feature engineering, and robust demand projections.
